# 环境

nvm 1.1.11

node 16.17.1

yarn 1.22.19



# 安装

nvm 安装

node安装  nvm install 16.17.1

node查看  nvm list

node切换  nvm use 16.17.1

yarn安装  npm install yarn

**依赖安装  yarn install**

**项目运行  npm run dev**

**项目打包  npm run build**
